What is the Document Object Modal (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the page structure as objects, allowing scripts to dynamically access, modify, and update content and structure.

how do you select on element from the DOM?

To select an element from the DOM, use methods like document.getElementById("id"), document.querySelector(".class" or "tag"), or document.getElementsByClassName("class"). These return references to matching elements.

what is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a parent element handles events for its child elements using a single event listener, improving performance and simplifying dynamic content management by reducing multiple listeners.

how do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute, getAttribute, or removeAttribute. For styles, use the style property, e.g., element.style.color = "red". You can also add/remove CSS classes with classList.add, classList.remove, or classList.toggle.
